# PRD

## 项目定位

个人技术博客，重点沉淀 AI、Agent、RPA、自动化、编程实践和项目复盘。

网站不是企业官网，也不是复杂产品系统，而是一个长期维护的个人技术知识库。

## 当前阶段

先做本地可用版本，优先完成：

- 首页有个人技术品牌感。
- 文章页适合长文阅读。
- 支持代码块、Callout、Mermaid 流程图和架构图。
- 项目记录能以卡片和 case study 形式展示。
- 移动端保持可读。

## 设计方向

采用 `tmp/design-mockups/clean-tech-journal-full/` 中的新模板方向：

```text
Clean Tech Journal
```

关键词：

- 干净
- 现代
- 技术感
- 内容优先
- 细边框卡片
- 深色代码块
- 浅色图表容器
- 少装饰、少动效

## 内容范围

主要内容：

- AI / Agent 实践
- RPA / 自动化流程
- 编程笔记和排错记录
- 项目记录和复盘
- Mermaid 流程图、架构图、框架图

## 非目标

当前不做：

- 评论
- 登录
- 后台
- 数据库
- CMS
- 在线画图编辑器
- 复杂动画
- 重型交互

## 技术方向

```text
Astro + Starlight + MDX + Mermaid
```

使用原因：内容型网站友好、静态构建简单、后续部署选择多、维护成本低。

## 开发环境

项目位于 WSL：

```bash
~/projects/personal-blog
```

开发、安装依赖、启动服务、构建验证都在 WSL 中执行。Windows 只用于浏览器预览。
